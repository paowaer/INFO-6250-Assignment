# Summary - Components

Components:
  - Functions that return HTML/JSX
  - Can be nested
  - Passed **props**
  - Must return single parent element or **fragment**
  - Must be named in **MixedCase**
  - **Requirements for this course**:
    - 1 component per `.jsx` file (must be `.jsx`)
    - Filename matches component name 
    - Semantic name
